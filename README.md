# bootreact-backend

Java 8, SpringBoot, MVC with HashMap,

# Run App
$ mvn spring-boot:run


# HTTP REQUESTS


**Get:**
$ http://localhost:8080/v1/api/computers

**Get: by id**
$ http://localhost:8080/v1/api/computers/1

**Delete: by id**
$ http://localhost:8080/v1/api/delete/1

**Post: by id**
$ http://localhost:8080/v1/api/create
